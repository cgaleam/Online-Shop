# Online-Shop

Este es un proyecto de tienda online desarrollado con un stack moderno, enfocado en una experiencia de usuario fluida y una gestión de datos en tiempo real.

---

## Tecnologías Utilizadas

* **Frontend:** React + Vite.
* **Estado Global:** React Context API
* **Backend & DB:** [Firebase](https://firebase.google.com/) (Authentication & Firestore).
* **Routing:** React Router DOM.

---

### Autenticación
* Registro e inicio de sesión de usuarios.
* Persistencia de sesión.

### Catálogo y Carrito
* Visualización dinámica de productos desde Firestore.
* Filtros por categorías o precios y buscador.
* Gestión de carrito y wishlist: añadir, eliminar y actualizar cantidades.
* Cálculo automático de totales.
